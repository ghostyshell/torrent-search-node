Features

Everything you need in a single self-hostable process.

  • 8 Torrent Sources

    Search Pirate Bay, 1337x, YTS, Nyaa.si, LimeTorrents, TorrentProject, TheHiddenBay, and Pornrips — simultaneously or individually.

  • Real-Debrid Integration

    Resolve magnet links to direct stream URLs. Per-user encrypted API keys, 20-hour TTL caching, and a 24-hour background refresh job for favorites.

  • Cloudflare Bypass

    Integrates with an upstream FlareSolverr instance for Cloudflare-protected sources. Gracefully skips if not configured.

  • Cover Image Caching

    Extract cover art from torrent descriptions. Store in MongoDB or an S3-compatible bucket with presigned URL auto-refresh.

  • 8 Background Jobs

    Autonomous cache-warming jobs run on configurable schedules: stream URL refresh, cover image pre-cache, Redis catalog pre-cache, log maintenance, and more.

  • Google OAuth + Favorites

    Session-based auth via Google OAuth. Per-user favorites, stored links, and torrent detail snapshots persisted in MongoDB.

Endpoints

Key REST endpoints. Full reference in api-reference.md.

Method Path Auth Description
GET /health — Liveness check
GET /api/torrents/search/:site/:query — Search a single source
POST /api/torrents/advanced-search — Multi-source search with filters
GET /api/torrents/browse/:category — Browse by category (no query)
GET /api/torrents/details/:site/:url — Fetch magnet + description + files
GET /api/auth/google — Initiate Google OAuth
POST /api/auth/exchange — Exchange one-time code for session
GET /api/storage/favorites required List user favorites
POST /api/auth/realdebrid/api-key required Store Real-Debrid API key
GET /api/monitoring/dashboard password Monitoring dashboard data

Quickstart

Up and running in under five minutes with Docker or Node.js directly.

Docker 
# Pull and run
docker build -t torrent-search-api .
docker run -p 3001:3001 \
  -e MONGODB_URI=mongodb://host:27017 \
  -e GOOGLE_SERVICE_ACCOUNT_JSON='...' \
  -e GOOGLE_CUSTOM_SEARCH_ENGINE_ID='...' \
  -e SESSION_SECRET='change-me' \
  -e FRONTEND_URL='http://localhost:3000' \
  torrent-search-api
Node.js 
# Clone and install
git clone https://github.com/akshatsinghkaushik/torrent-search-node.git
cd torrent-search-node
npm install

# Configure
cp .env.example .env
# Edit .env with your MongoDB URI, Google creds, SESSION_SECRET

# Start
npm start
Sample requests

# Search Pirate Bay

curl "http://localhost:3001/api/torrents/search/piratebay/ubuntu/1"

curl -X POST "http://localhost:3001/api/torrents/advanced-search" \

-H "Content-Type: application/json" \

# Health check

curl "http://localhost:3001/health"

FAQ

What torrent sources are supported?
Eight sources: The Pirate Bay, 1337x (via FlareSolverr), YTS, Nyaa.si, LimeTorrents, TorrentProject, TheHiddenBay, and Pornrips. Adding new sources requires a scraper module and a one-line registration in the service registry.
What do I need to self-host?
Required: Node.js 22+, a MongoDB instance, Google Cloud credentials (service account + OAuth client).
Optional: Redis (catalog pre-cache), an S3-compatible bucket (cover images), and a FlareSolverr instance (1337x scraper).
How does Real-Debrid integration work?
Each authenticated user can store their own Real-Debrid API key (encrypted with AES-256-GCM, key derived from your SESSION_SECRET). The API resolves magnet links to direct HTTP stream URLs and caches them for 20 hours. A background job runs every 24 hours to proactively refresh URLs for all favorited magnets across all users.
Does the search API require authentication?
No. The search, browse, and detail endpoints are open. Authentication is only required for personal data: favorites, stored links, and Real-Debrid key management. You can also restrict access to these public endpoints using rate limiting and CORS configuration.
What is FlareSolverr and is it required?
FlareSolverr is a self-hosted proxy that solves Cloudflare challenges using a headless browser. It is only needed for the 1337x scraper. If FLARESOLVERR_URL is not set, the 1337x scraper returns an error while all other scrapers continue to function normally.
